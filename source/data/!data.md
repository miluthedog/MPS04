# Naming convention

### Data type
- raw: raw data
- cyc: invidual cycle
- rmi: remove idle
### Workpiece
- b: black, drop on slider
- w: white, drop on conveyor
### Error
- E1: vertical axis vibration
- E2: horizontal axis vibration
